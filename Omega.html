HTML Linguagem de Marcação de hipertexto

Tutorias para iniciantes

HTML(Linguagem de Marcação de HiperTexto) é o bloco de construção mais basico da web. Define o significado e a estrutura do conteudo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparencia/apresentação(CSS) ou a funcionalidade/comportamento (JavaScript) de uma pagina da web.

TAG UTILIDADE
b é usada para aplicar estilo de negrito ao texto dentro dela
mark é usada para destacar ou realçar um trecho de texto
ins é usada para indicar um texto inserido ou adicionado em relação a uma versão anterior de um documento
strong é usada para aplicar estilo de forte importância ao texto dentro dela
i é usada para aplicar estilo de itálico ao texto dentro dela
small é usada para indicar que o texto dentro dela é de tamanho pequeno em relação ao texto ao redor
del é usada para indicar texto que foi removido ou deletado de uma versão anterior de um documento
sup é usada para criar um texto sobrescrito, que geralmente é menor e posicionado acima da linha de base normal do texto

Protocolos Web

  • HTTP (Hypertext Transfer Protocol): é o protocolo padrão para transferir documentos hipertexto, como páginas da web, na Internet. É o protocolo principal usado para comunicação entre um navegador e um servidor web.
  • HTTPS (Hypertext Transfer Protocol Secure): é a versão segura do HTTP, que usa uma camada de criptografia SSL/TLS para proteger a comunicação entre o navegador e o servidor. É amplamente utilizado para transações financeiras, comércio eletrônico e outras atividades sensíveis à segurança.
  • FTP (File Transfer Protocol): é um protocolo de rede usado para transferir arquivos entre um cliente e um servidor. É usado principalmente para transferência de arquivos grandes, como imagens, vídeos e arquivos de software.
  • SMTP (Simple Mail Transfer Protocol): é o protocolo padrão usado para enviar e-mails na Internet. Ele permite que um servidor de e-mail envie mensagens para outro servidor de e-mail e, em seguida, o destinatário possa receber a mensagem.
  • WebSocket: é um protocolo de rede que permite comunicação bidirecional em tempo real entre um navegador e um servidor. É amplamente usado para aplicativos de bate-papo, jogos online e outras aplicações que requerem comunicação em tempo real.
